Algorithm: Polynomial Addition Using Array of Structures

Step 1: Start

Step 2: Declare a structure containing: • Coefficient (coeff) • Exponent
(expo)

Step 3: • x1[] → First polynomial • x2[] → Second polynomial • x3[] →
Resultant polynomial

Step 4: Read the number of terms m and n of the first and second
polynomials.

Step 5: Read the coefficients and exponents of both polynomials in
descending order of exponents.

Step 6: Initialize the indices: i ← 0 j ← 0 k ← 0

Step 7: Repeat while (i < m) and (j < n).

Case 1: Exponents are equal If x1[i].expo = x2[j].expo then x3[k].coeff
← x1[i].coeff + x2[j].coeff x3[k].expo ← x1[i].expo i ← i + 1 j ← j + 1
k ← k + 1

Case 2: First polynomial exponent is greater Else if x1[i].expo >
x2[j].expo then x3[k].coeff ← x1[i].coeff x3[k].expo ← x1[i].expo i ←
i + 1 k ← k + 1

Case 3: Second polynomial exponent is greater Else x3[k].coeff ←
x2[j].coeff x3[k].expo ← x2[j].expo j ← j + 1 k ← k + 1

Step 8: If the first polynomial has remaining terms, While (i < m)
x3[k].coeff ← x1[i].coeff x3[k].expo ← x1[i].expo i ← i + 1 k ← k + 1

Step 9: If the second polynomial has remaining terms, While (j < n)
x3[k].coeff ← x2[j].coeff x3[k].expo ← x2[j].expo j ← j + 1 k ← k + 1

Step 10: Display all the terms stored in x3[] as the resultant
polynomial.

Step 11: Stop.
